# Bank Management System

A simple yet powerful **Bank Management System** built with C++ and Visual Studio.  
This project demonstrates core concepts of **Object-Oriented Programming (OOP)**, secure authentication, and transaction handling in a banking environment.

---

## ✨ Features

- **Login System**
  - Secure login screen for users.
  - Passwords are stored using **encryption and decryption** methods for safety.

- **Client Management**
  - Add new clients.
  - View client list.
  - Update client information.
  - Delete clients.
  - Search for clients.

- **Transactions**
  - Deposit money.
  - Withdraw money.
  - Transfer between accounts.
  - View total balances.

- **User & Security Management**
  - Manage system users.
  - Track login history.
  - Maintain transfer logs.
